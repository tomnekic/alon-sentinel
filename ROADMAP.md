# Roadmap

This roadmap reflects current thinking and is subject to change.
Community feedback shapes priorities.

## Shipped

- HTTP, SSL, TCP, DNS, Heartbeat monitor types
- Incident tracking with acknowledgement
- Webhook, Slack, Discord, email notifications
- Full RBAC and JWT-based auth
- API-first design with OpenAPI spec
- React admin UI
- Docker Compose deployment
- Public status pages
- Prometheus metrics (gauges, histograms, per-type counters)

## Near-term (planned)

- Docker image and registry publishing
- Improved check history visualization
- Bulk monitor operations

## Considering

- Custom domain status pages
- SSO / OIDC support
- SLA and SLO reporting
- Multi-region check workers
- PagerDuty / OpsGenie integration
- SMS notifications
- Escalation policies
- Advanced audit logs
- HA deployment tooling

## Not planned (yet)

- Built-in APM or tracing
- Log aggregation
- Native mobile app
